# Personal Introduction Program (Week 1 – Python Basics)

## Project Overview
This is a simple Python program created as part of the Week 1 internship task.  
The program collects basic user information such as name, age, city, and hobby, and displays a friendly welcome message.

## Program Description
- Takes user input using the `input()` function
- Stores the inputs in variables
- Displays a personalized welcome message using `print()` and f-strings

## Files Included
- `personal_intro.py` – Python source code
- `requirements.txt` – List of dependencies
- `screenshot.png` – Sample program output
